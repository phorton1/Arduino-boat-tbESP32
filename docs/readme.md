**tbESP32** is an ESP32-based WiFi bridge designed to plug into the GP8
general-purpose connector on the teensyBoat board.  It provides TCP/IP and UDP
remote access to the teensyBoat firmware, allowing the companion wxPerl application
(**[teensyBoat.pm](https://github.com/phorton1/base-apps-teensyBoat/blob/master/docs/readme.md)**)
to connect over a local WiFi network rather than a direct USB serial cable.

Full documentation is planned.
